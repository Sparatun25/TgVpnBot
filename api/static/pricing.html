Бесплатный триал

Каждый новый пользователь получает 3 дня бесплатного доступа для тестирования сервиса. Триал предоставляется один раз на аккаунт.

Оплата

Оплата производится через систему быстрых платежей (СБП). Средства зачисляются на баланс аккаунта и списываются при покупке тарифа. Возврат средств возможен в течение 24 часов при технической проблеме.

Продление

Подписка продлевается автоматически при наличии средств на балансе. Вы можете отключить автопродление в настройках аккаунта.
